# Discover AI Skills — Claude Plugin

Search and discover 36,000+ AI agent skills, MCP servers, and tools directly from Claude.

## What This Plugin Does

Adds a `discover-ai-skills` skill to Claude that lets you search [discoveraiskills.com](https://discoveraiskills.com) by keyword, category, or use case — without leaving your AI session.

## Install

```bash
claude plugin install https://github.com/roach54023/agent-skills/tree/main/claude-plugin
```

## Usage

Just ask Claude naturally:

- *"Find me a skill for web scraping"*
- *"Is there a Claude skill for stock analysis?"*
- *"What MCP servers are available for databases?"*
- *"Search for PDF processing skills"*

Claude will search discoveraiskills.com and return the best matches with direct install links.

## About DiscoverAISkills

[discoveraiskills.com](https://discoveraiskills.com) is the largest community catalog for AI agent skills — covering Claude Code skills, MCP servers, OpenClaw skills, and Cursor extensions. Updated daily from GitHub and community submissions.

- 36,000+ skills indexed
- Free to browse and install
- Submit your own: [discoveraiskills.com/submit](https://discoveraiskills.com/submit)
