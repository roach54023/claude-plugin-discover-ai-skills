# Discover AI Skills — Claude Plugin

This plugin adds the `discover-ai-skills` skill to Claude Code, enabling search and discovery of 36,000+ AI agent skills directly from your AI session.

## What It Does

When you ask Claude to find a skill, tool, or MCP server, this plugin searches [discoveraiskills.com](https://discoveraiskills.com) and returns the best matches with direct install links.

## Usage

Just ask naturally:
- "Find me a skill for web scraping"
- "Is there a Claude skill for stock analysis?"
- "What MCP servers are available for databases?"
- "Search for PDF processing skills"

## Skills Included

- **discover-ai-skills** — Search discoveraiskills.com by keyword, category, or use case

## About

[discoveraiskills.com](https://discoveraiskills.com) is the largest community catalog for AI agent skills — covering Claude Code skills, MCP servers, OpenClaw skills, and Cursor extensions. 36,000+ skills indexed, updated daily.
